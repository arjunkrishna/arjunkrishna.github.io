Unable to remove node-modules folder from Windows machine.

2016-09-24
npm

original post: http://t.co/0pV4OSTG1L

Install rimraf (https://www.npmjs.com/package/rimraf is a deep deletion module)

npm install rimraf -g

within the project folder run the following command to remove the folder

rimraf node_modules
